## Integration Guide

大多数应用只需配置以下信息即可连接到 Model Router：

1. **API 地址**：你的 Model Router 服务地址
2. **API Key**：在 Model Router 控制台生成的 Token
3. **模型名称**：选择要使用的模型

具体配置步骤，请参考各应用的详细文档。
