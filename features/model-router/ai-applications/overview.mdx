## Integration Guide

Most applications only need to configure the following information to connect to Model Router:

1. **API Address**: Your Model Router service address
2. **API Key**: Token generated in the Model Router console
3. **Model Name**: Select the model to use

For specific configuration steps, refer to each application's detailed documentation.
