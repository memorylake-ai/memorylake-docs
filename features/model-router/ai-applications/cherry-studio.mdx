## Model Router Integration Method

### Parameter Configuration

The following information is required to configure Cherry Studio to connect to Model Router:

- **Provider Type**: Select a type supported by Model Router
- **API Key**: API Key obtained from the Model Router [console](https://app.memorylake.ai/panel/token)
- **API Address**: Model Router service address

## Quick Configuration

<Info>
In the chat settings under system settings in the Model Router [console](https://app.memorylake.ai/panel), you can add the following quick option to one-click fill Cherry Studio configuration on the [token management page](https://app.memorylake.ai/panel/token).
</Info>

```json
{ "Cherry Studio": "cherrystudio://providers/api-keys?v=1&data={cherryConfig}" }
```
