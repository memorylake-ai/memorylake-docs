## Model Router 集成方法

### 参数配置

配置 Cherry Studio 连接到 Model Router 需要以下信息：

- **Provider Type**：选择 Model Router 支持的类型
- **API Key**：从 Model Router 控制台获取的 API Key
- **API Address**：Model Router 服务地址

### 可视化指南

#### 1. 复制 API Key

在 Model Router 控制台的令牌管理页面，复制你的 API Key。

## 快捷配置

<Info>
在 Model Router 控制台的系统设置中的聊天设置，可以添加以下快捷选项，在令牌管理页面一键填充 Cherry Studio 配置。
</Info>

```json
{ "Cherry Studio": "cherrystudio://providers/api-keys?v=1&data={cherryConfig}" }
```
