## 快速认识

Model Router 提供统一的模型路由与多模态任务入口，兼容主流协议与通道。

- 一个入口覆盖主流模型与任务，兼容 OpenAI 风格调用，无需改 SDK
- 同时保留 Claude、Gemini 等原生路径，减少迁移和切换成本
- 内置稳定性、配额结算、通知与安全策略，帮助你在生产环境放心接入

## 快速开始

1) 获取令牌：在控制台生成令牌，确保关联的分组已开通所需模型与额度。  
2) 查看可用模型：调用 `/v1/models` 或对应原生路径，确认当前令牌可用的模型与通道。  
3) 选择入口：沿用 OpenAI 风格 SDK 直接改 base URL，或使用 Claude、Gemini 原生路径。  
4) 发送请求：按模型或任务接口的要求传参，替换 `Authorization` 与目标模型即可调通。
