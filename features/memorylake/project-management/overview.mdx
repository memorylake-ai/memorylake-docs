## Overview

Projects help you organize related documents and memories into cohesive units. Provide structured context for AI interactions and team collaboration.

## What is a Project?

A project in MemoryLake is:
- **Container** for related documents
- **Context provider** for AI models via MCP servers
- **Organization unit** for knowledge management
- **Collaboration space** for teams

## Project Components

### Documents
Associate files and folders from your drive. Documents provide content that AI models can read and analyze.

### Memories
Add knowledge snippets and context that guide AI understanding of your project.

### MCP Servers
Generate API keys for external applications and AI models to access project data.
