## Creating Projects

<Steps>
<Step title="Navigate to Projects">
Go to the Project Management section.
</Step>

<Step title="Click Create Project">
Click the "Create Project" button.

{/* SCREENSHOT: Create project button */}
</Step>

<Step title="Enter Details">
Fill in:
- **Name** (required): Descriptive project name
- **Description** (optional): Explain project purpose

{/* SCREENSHOT: Create project dialog */}
</Step>

<Step title="Save">
Click "Create" or "Save" to create the project.

<Check>
Your project appears in the projects list.
</Check>
</Step>
</Steps>

## Viewing Projects

### Project List

All projects appear in a paginated list:
- Project name and description
- Document count
- Memory count
- MCP server count
- Created/updated timestamps

{/* SCREENSHOT: Projects list with data */}

### Pagination

Navigate through projects:
- Items per page: 10, 25, 50, or 100
- Previous/Next buttons
- Jump to specific page

## Editing Projects

### Inline Editing

From project dashboard:
1. Click project name to edit
2. Type new name
3. Press Enter to save or Escape to cancel

{/* SCREENSHOT: Inline editing mode */}

### Edit Dialog

From project list:
1. Click edit icon or "Edit" in actions menu
2. Modify name and/or description
3. Click "Save"

## Deleting Projects

<Warning>
Deleting a project is permanent and removes all associations. Documents and memories in your drive are not deleted, only the project container.
</Warning>

<Steps>
<Step title="Select Project">
From project list, click delete icon or select "Delete" from actions.

{/* SCREENSHOT: Delete confirmation dialog */}
</Step>

<Step title="Confirm Deletion">
Review project name and click "Delete" to confirm.
</Step>

<Step title="Verify Removal">
Project disappears from list. Associated documents remain in your drive.
</Step>
</Steps>
