## Common Issues

<AccordionGroup>

<Accordion title="Cannot create project">
**Solutions**:
- Check project name isn't empty
- Ensure name isn't too long (< 255 chars)
- Verify internet connection
- Check browser console for errors
- Retry after refreshing page
</Accordion>

<Accordion title="Documents won't associate with project">
**Solutions**:
- Verify documents selected in dialog
- Check processing status for errors
- Ensure files are supported types
- Try associating fewer files at once
- Check project hasn't reached document limit
</Accordion>

<Accordion title="Document status shows 'error'">
**Solutions**:
- Hover over error icon for details
- Verify file still exists in drive
- Check file isn't corrupted
- Remove and re-add document
- Contact support with error message
</Accordion>

<Accordion title="Project dashboard not loading">
**Solutions**:
- Refresh page
- Clear browser cache
- Check internet connection
- Try different browser
- Verify project still exists
</Accordion>

<Accordion title="Inline editing not saving">
**Solutions**:
- Press Enter to confirm (not just click away)
- Check for error message
- Verify new name is valid
- Refresh and check if change persisted
- Use edit dialog instead
</Accordion>

</AccordionGroup>
