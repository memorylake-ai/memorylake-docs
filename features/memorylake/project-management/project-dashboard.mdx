## Dashboard Overview

The project dashboard provides a comprehensive view of your project with statistics, documents, memories, and MCP servers.

{/* SCREENSHOT: Complete project dashboard */}

## Statistics Cards

View key metrics at a glance:

**Document Count**: Number of associated documents

**Memory Count**: Number of memories

**MCP Server Count**: Number of active API keys

{/* SCREENSHOT: Statistics cards section */}

## Inline Editing

Edit project details directly from dashboard:

1. Click project name - becomes editable
2. Type new name
3. Press Enter to save

Same for description field.

{/* SCREENSHOT: Inline editing interface */}

## Tab Navigation

Dashboard has three tabs:

### Documents Tab
- List of associated documents
- Document status (pending, running, okay, error)
- Add/remove documents
- View processing errors

### Memories Tab
- List of project memories
- Create/edit/delete memories
- View memory content

### MCP Servers Tab
- List of API keys
- Generate new keys
- View endpoint URLs
- Delete keys

{/* SCREENSHOT: Tab navigation */}

## Quick Actions

From dashboard you can:
- Add documents
- Create memories
- Generate API keys
- Edit project details
- Return to project list
