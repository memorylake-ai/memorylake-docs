## Common Issues

<AccordionGroup>

<Accordion title="Cannot create memory">
**Solutions**:
- Ensure name field is filled
- Check content isn't empty
- Verify name isn't too long
- Check internet connection
- Refresh page and retry
</Accordion>

<Accordion title="Memory not saving">
**Solutions**:
- Wait for save confirmation
- Check for error message
- Verify content length reasonable (< 100KB)
- Check network connection
- Copy content as backup, refresh, and retry
</Accordion>

<Accordion title="Cannot find memory after creating">
**Solutions**:
- Verify you're viewing correct project
- Check Memories tab (not Documents)
- Refresh the page
- Use browser search to find by name
- Check if memory was actually saved (error may have occurred)
</Accordion>

<Accordion title="Memory content truncated in list">
**This is normal**:
- List view shows only first 3 lines as preview
- Click memory name to view full content
- Full content preserved, just not shown in list
</Accordion>

<Accordion title="Cannot edit memory">
**Solutions**:
- Check you have edit permissions
- Verify memory still exists
- Refresh page and try again
- Check browser console for errors
- Try different browser
</Accordion>

<Accordion title="Delete confirmation not appearing">
**Solutions**:
- Allow popups for MemoryLake
- Check popup blocker settings
- Try different browser
- Use keyboard Delete key
- Refresh page and retry
</Accordion>

</AccordionGroup>

## Getting Help

If issues persist, contact support with:
- Memory name and project name
- What you were trying to do
- Error messages (if any)
- Browser and version
- Screenshots if applicable
