## Overview

Memories are knowledge snippets that provide context for your projects. Store notes, guidelines, and information that helps both humans and AI understand your project.

## What Are Memories?

Memories are:
- **Text-based** knowledge snippets
- **Project-scoped** contextual information
- **AI-accessible** via MCP servers
- **Searchable** and manageable

## Memory Features

- **Name & Content**: Each memory has a descriptive name and text content
- **Timestamps**: Track when memories were created and updated
- **Full CRUD**: Create, read, update, and delete operations
- **Project Association**: Memories belong to specific projects
- **MCP Access**: AI models can read memories via MCP servers

## Getting Started

<Steps>
<Step title="Open Project">
Navigate to a project dashboard.
</Step>

<Step title="Go to Memories Tab">
Click the "Memories" tab.
</Step>

<Step title="Create Memory">
Click "Add Memory" and fill in name and content.
</Step>

<Step title="Save">
Save your memory. It's now available to AI models accessing the project.
</Step>
</Steps>
