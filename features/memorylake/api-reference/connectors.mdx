## Check Connector Status

```
GET /connector/status/{provider}
```

<ParamField path="provider" type="string" required>
Connector provider (e.g., "wps", "dropbox")
</ParamField>

Returns connection status and available libraries.

## Login to Connector

```
GET /connector/login/{provider}
```

Initiates OAuth flow. Returns authorization URL.

## List WPS Document Libraries

```
GET /connector/wps/doclibs
```

Returns available WPS document libraries for connection.

## Revoke Connector Authorization

```
DELETE /connector/revoke/{provider}
```

Revokes authorization and removes all mount points.

## Remove Connection

```
DELETE /connector/{provider}/doclibs/{driveId}/connections
```

Removes specific document library connection.
