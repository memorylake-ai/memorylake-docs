## Get User Drive

Retrieve the authenticated user's drive information.

```
GET /drives
```

### Headers

<ParamField header="Authorization" type="string" required>
Bearer token for authentication
</ParamField>

### Response

<ResponseField name="success" type="boolean" required>
Indicates if the request succeeded
</ResponseField>

<ResponseField name="data" type="object">
Drive object

<Expandable title="Drive properties">
  <ResponseField name="id" type="string">
  Unique drive identifier
  </ResponseField>

  <ResponseField name="root_item_id" type="string">
  ID of the root folder
  </ResponseField>

  <ResponseField name="created_at" type="timestamp">
  Drive creation timestamp
  </ResponseField>
</Expandable>
</ResponseField>

### Example

<RequestExample>
```bash cURL
curl -X GET 'https://app.memorylake.ai/api/memorylake/api/v1/drives' \
  -H 'Authorization: Bearer YOUR_API_KEY'
```
</RequestExample>

<ResponseExample>
```json Success (200)
{
  "success": true,
  "data": {
    "id": "drive_123",
    "root_item_id": "item_root_456",
    "created_at": "2024-01-15T10:00:00Z"
  }
}
```
</ResponseExample>

## Create Drive

Initialize a new drive for the user.

```
POST /drives
```

### Response

Same as GET /drives
