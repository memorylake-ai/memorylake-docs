## 平台概览

MemoryLake AI 提供统一的模型路由与多模态任务入口，兼容主流协议与通道，帮助你在生产环境中高效、安全地接入 AI 能力。

## 核心能力

- **统一入口**：兼容 OpenAI、Claude、Gemini 等主流协议，无需修改现有客户端代码
- **智能路由**：自动选择最优通道，支持重试与容错，保障服务稳定性
- **精细控制**：令牌额度管理、配额分配、使用统计，满足企业级需求
- **团队协作**：支持多人协作、角色权限、邀请管理，适合团队使用

## 开始使用

1) **获取令牌**：在控制台生成 API Key，设置额度与到期时间。  
2) **查看文档**：选择 Model Router 或 Team Collaboration 开始探索。  
3) **首次调用**：参考 [Model Router 文档](/features/model-router/overview) 完成首次 API 调用。  
4) **深入了解**：查看各功能模块的详细文档，了解更多高级功能。
